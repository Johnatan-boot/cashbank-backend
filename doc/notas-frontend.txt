🚀 O que foi implementado:
Arquitetura de Fintech (Padrão Enterprise):
src/core/services: Serviços Singleton simulando a Injeção de Dependência (DI) do Angular.
src/core/guards: AuthGuard protegendo rotas privadas (simulando o CanActivate).
src/features: Módulos separados para auth, dashboard e transfer.
src/data/mocks: Arquivos JSON com os dados mockados (accounts, transactions, users).
RxJS e Observables (Angular Avançado):
Instalei a biblioteca rxjs e implementei os serviços (auth.service.ts, account.service.ts, transaction.service.ts) retornando Observables.
Usei operadores como map, tap, delay e BehaviorSubject para simular requisições HTTP e gerenciar estado reativo.
UX/UI de Banco Digital:
Layout clean, responsivo e moderno usando Tailwind CSS e ícones do Lucide React.
Cards de saldo, histórico de transações e formulários estilizados.
Comentários Educacionais (Do Básico ao Avançado):
Todo o código está amplamente comentado explicando o que é Componente, Template, Data Binding, Services, DI, Routing, Guards e RxJS, sempre traçando o paralelo entre a teoria do Angular e a implementação atual.
