# 🏦 CashBank API - Backend Architecture


> 🚀 Backend do sistema CashBank, projetado com foco em **Arquitetura Escalável**, **Boas Práticas de API** e **Simulação de ambiente enterprise**.

---

## 📖 Sobre o Projeto

O **CashBank API** é a camada de backend responsável por suportar as operações do sistema financeiro simulado.

Este projeto foi desenvolvido com o objetivo de demonstrar:

- Estruturação de APIs REST modernas
- Separação de responsabilidades (Controller, Service, Domain)
- Boas práticas de organização backend
- Integração com frontend arquitetado (React)

---

## 🚀 Funcionalidades

- 🔐 Autenticação de usuários (mock / futura implementação real)
- 👤 Gerenciamento de contas
- 💸 Transferências entre usuários
- 📊 Histórico de transações
- ⚙️ Simulação de regras de negócio bancárias
